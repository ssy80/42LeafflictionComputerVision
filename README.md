# Leaffliction

*This project has been created as part of the 42 Advanced curriculum by ssian and axlee.*

## Description

Leaffliction is a machine learning project that classifies plant leaf diseases from images. Given a photo of an apple or grape leaf, the model identifies which disease (if any) is present across 8 classes.

The pipeline covers the full ML workflow:

* **Distribution** — visualise the class balance of the raw dataset, saved as charts to `distribution/before/` and `distribution/after/`.
* **Augmentation** — expand the dataset by applying 6 transformations per image (flip, rotate, skew, crop, distortion, contrast), growing each class from ~100 to ~700 images. Output saved to `dataset/augmented/`.
* **Transformation** — apply computer vision preprocessing (Gaussian blur, mask, ROI, analyze, pseudolandmarks) to prepare images for training.
* **Training** — train separate CNNs for Apple and Grape (TensorFlow/Keras) with early stopping and TensorBoard logging. Models saved to `models/Apple/splited/` and `models/Grape/splited/`.
* **Prediction** — classify a single leaf image and display the original alongside its masked transformation and predicted disease label.

### Dataset Classes

| Apple | Grape |
|---|---|
| Apple\_Black\_rot | Grape\_Black\_rot |
| Apple\_healthy | Grape\_Esca |
| Apple\_rust | Grape\_healthy |
| Apple\_scab | Grape\_spot |

## Instructions

### Setup

Place your dataset images into `dataset/raw/` following the folder structure above, then run:

```bash
source scripts/setup.sh
```

> On machines with an NVIDIA GPU, `setup.sh` automatically installs CUDA-enabled TensorFlow and configures the library paths. On CPU-only machines (e.g. school PCs) it installs plain TensorFlow.

To tear down the environment:

```bash
source scripts/remove.sh
```

---

## Order of Execution

```bash
source scripts/setup.sh      # 1. set up environment
make distribution            # 2. analyse raw dataset balance
make augment                 # 3. expand dataset
make distribution-after      # 4. verify balance after augmentation 
make transform               # 5. apply transformations
make train                   # 6. train Apple and Grape models
make predict-apple           # 7. predict a random Apple test image
make predict-grape           # 8. predict a random Grape test image
```

## Makefile Commands

| Command | Description |
|---|---|
| `make distribution` | Analyse raw dataset → saves charts to `distribution/before/` |
| `make distribution-after` | Analyse augmented dataset → saves charts to `distribution/after/` |
| `make augment` | Augment all raw images → `dataset/augmented/` |
| `make transform` | Apply transformations to all augmented images → `dataset/transformed/` |
| `make train` | Train Apple and Grape models → `models/Apple/splited/` and `models/Grape/splited/` |
| `make predict` | Predict a random Apple test image (uses `IMG` variable) |
| `make predict-apple` | Predict a random Apple test image using the Apple model |
| `make predict-grape` | Predict a random Grape test image using the Grape model |
| `make test` | Run the full pytest suite |
| `make lint` | Run flake8 linter on `src/` |
| `make logs` | Launch TensorBoard at `http://localhost:6006` |
| `make clean` | Remove models, logs, augmented data, distribution charts, and caches |

## Step-by-Step Details

### 1. Distribution

Analyse the raw dataset class balance before and after augmentation. Charts are saved as PNG files — no window opens.

```bash
make distribution        # → distribution/before/Apple/ and distribution/before/Grape/
make distribution-after  # → distribution/after/Apple/  and distribution/after/Grape/
```

Raw commands:

```bash
python3 -m distribution.distribution dataset/raw/Apple/   distribution/before/Apple
python3 -m distribution.distribution dataset/raw/Grape/   distribution/before/Grape
python3 -m distribution.distribution dataset/augmented/Apple/ distribution/after/Apple
python3 -m distribution.distribution dataset/augmented/Grape/ distribution/after/Grape
```

Expected result:
- **Before**: ~100 images per class
- **After**: ~700 images per class (7× increase, all classes balanced)

---

### 2. Augmentation

Expand the dataset by generating 6 augmented variants per image. The 6 augmentations applied per image: **Flip**, **Rotate**, **Skew**, **Contrast**, **Crop**, **Distortion**.

```bash
make augment
```

---

### 3. Transformation

Apply computer vision preprocessing to all augmented images. Produces 6 transformed variants per image (original, gaussian_blur, mask, roi, analyze, pseudolandmarks).

```bash
make transform
```

Raw commands:

```bash
python3 -m transformation.transformation -src dataset/augmented/Apple/Apple_Black_rot -dst dataset/transformed/Apple/Apple_Black_rot
python3 -m transformation.transformation -src dataset/augmented/Apple/Apple_healthy   -dst dataset/transformed/Apple/Apple_healthy
python3 -m transformation.transformation -src dataset/augmented/Apple/Apple_rust      -dst dataset/transformed/Apple/Apple_rust
python3 -m transformation.transformation -src dataset/augmented/Apple/Apple_scab      -dst dataset/transformed/Apple/Apple_scab
python3 -m transformation.transformation -src dataset/augmented/Grape/Grape_Black_rot -dst dataset/transformed/Grape/Grape_Black_rot
python3 -m transformation.transformation -src dataset/augmented/Grape/Grape_Esca      -dst dataset/transformed/Grape/Grape_Esca
python3 -m transformation.transformation -src dataset/augmented/Grape/Grape_healthy   -dst dataset/transformed/Grape/Grape_healthy
python3 -m transformation.transformation -src dataset/augmented/Grape/Grape_spot      -dst dataset/transformed/Grape/Grape_spot
```

To preview a single image (displays all transformations interactively):

```bash
python3 -m transformation.transformation -src "dataset/raw/Apple/Apple_Black_rot/image (1).JPG"
```

---

### 4. Training

Train separate CNN models for Apple and Grape.

```bash
make train
```

Raw commands:

```bash
python3 -m training.train dataset/augmented/Apple models/Apple
python3 -m training.train dataset/augmented/Grape models/Grape
```

---

### 5. Prediction

Classify a leaf image and display the original alongside its masked transformation.

```bash
make predict-apple   # picks a random image from dataset/test/Apple/
make predict-grape   # picks a random image from dataset/test/Grape/
```

Override with a specific image:

```bash
make predict-apple APPLE_IMG="dataset/test/Apple/image (5).JPG"
make predict-grape GRAPE_IMG="dataset/test/Grape/image (3).JPG"
```

Raw commands:

```bash
python3 -m prediction.predict "dataset/test/Apple/image (1).JPG" models/Apple/splited
python3 -m prediction.predict "dataset/test/Grape/image (1).JPG" models/Grape/splited
```

Output: displays the original image alongside its masked transformation, and prints the predicted class label to stdout.
